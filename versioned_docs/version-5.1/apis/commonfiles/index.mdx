### db/ {/* #db */}

The `db/` directory contains files related to database schema and data management, such as installation, upgrade, access control, scheduled tasks, and events.

:::caution

Files inside the `db/` folder (such as `install.xml`, `upgrade.php`, `access.php`, `tasks.php`, etc.) must **never** contain `require_once` (or any other file inclusion). These files are parsed by Moodle in specific, performance-sensitive contexts and must remain self-contained definition files with no side-effects.

:::
